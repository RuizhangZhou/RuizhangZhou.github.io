Automatic Speech Recognition System

RWTH Software Project | ASR Research | 2022-2023

Tech Stack
C++ HMM Neural Networks Signal Processing
Research Overview

Implemented custom ASR system from scratch using both traditional HMM-based methods and neural network approaches. Gained deep understanding of acoustic modeling, language modeling, and decoding algorithms.

Components
  • Feature Extraction: MFCC, filterbank features for audio representation
  • Acoustic Modeling: HMM-GMM and DNN-HMM hybrid systems
  • Language Modeling: N-gram models for linguistic constraints
  • Decoding: Viterbi algorithm for optimal sequence prediction

Master Thesis: AI-Based Scenario Generation for Autonomous Vehicles

RWTH Cyber-Physical Mobility Group | Oct 2023 - Apr 2024

Tech Stack
Python PyTorch GAN Diffusion Models Reinforcement Learning
Research Contribution

Developed novel generative AI approaches for synthesizing realistic and diverse testing scenarios for motion planners in connected and automated vehicles. Combined GAN and Diffusion Model architectures to generate edge cases that reveal system vulnerabilities.

Research Methodology
  • Formulated scenario generation as conditional generative modeling problem
  • Implemented and compared GAN vs Diffusion Model approaches
  • Designed evaluation metrics for scenario realism and test coverage
  • Validated generated scenarios using motion planning algorithms
  • Analyzed failure modes and edge case discovery capabilities
Research Impact
✓ Automated generation of diverse test scenarios, reducing manual effort
✓ Discovered edge cases not covered by traditional rule-based methods
✓ Improved test coverage for autonomous vehicle safety validation
